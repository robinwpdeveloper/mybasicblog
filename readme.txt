Baisc WordPress Theme

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Basic Theme in the search form and press the 'Enter' key in your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Go to mybasicblog.com for a guide to customize this theme.
5. Navigate to Appearance > Customize in your admin panel.
